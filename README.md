# local-wifi-viewer
本机WiFi密码查看工具
## 功能亮点
- 支持查看本机所有已连接的WiFi密码（去除密码为空的WiFi）
- 支持将WiFi密码一键复制到剪贴板
- 支持根据WiFi名称查询
- 支持右键删除已连接的WiFi
- 默认将当前连接的WiFi置顶显示
